# PyChat

## How to run
python chat.py {username}
- username: the username you want to use

## Guide
- Transfer files by typing "transfer {filename}" and press enter.
- Each user chooses which port to use. The port number is displayed on the screen.
- When you wish to exit the chat, type "exit" and press enter. Both users must type exit to exit the chat.
- Normal chatting is done by typing and pressing enter.
